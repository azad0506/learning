how to remove master/main branch

M mean forcefully

  • git branch -m branchname
  • git branch -M branchname

how to create a branch

  • git branch branchname/feature

how to check our branch is created or not

  • git branch

to move from one branch to other branch

  • git switch feature/navbar

to merge a branch

  • git merge branchname

delete branch

  • git branch -d branchname
